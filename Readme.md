								ROCK PAPER SCISSOR v0.1 Alpha Release

To run the game of "Rock Paper Scissior", you must have the following dependencies installed:

- Python (v2.7)
- Pygame (>v1.9.2) 
- openCV (v3.0.0)
- pyttsx

To execute the game:
run main.py in linux using python main.py
